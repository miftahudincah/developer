📘 Panduan Sistem Absensi

Sistem menghubungkan Web dengan ESP32 via Firebase

1. Pendaftaran Siswa
Langkah-langkah:
- Guru input data siswa di menu Data Siswa (FP)
- Generate kode registrasi di menu Manajemen Pengguna
- Siswa daftar dengan kode dan ID Fingerprint
2. Absensi Fingerprint
Cara absen:
- Scan fingerprint di sensor ESP32 untuk absen MASUK
- Scan fingerprint lagi untuk absen PULANG (setelah delay)
- Data otomatis masuk ke tabel absensi
3. Pengaturan Kelas & Jurusan
Manajemen Kelas:
- Admin dapat menambah/edit kelas sesuai kebutuhan (VII A, VIII B, dll)
- Untuk SMK, dapat juga menambah jurusan (RPL, TKJ, dll)
- Semua perubahan akan otomatis tersimpan di Firebase
4. Pengumuman
Fitur pengumuman:
- Admin/Guru dapat membuat pengumuman dengan batas waktu
- Pengumuman muncul real-time untuk semua pengguna
- Notifikasi akan muncul saat ada pengumuman baru
5. Rekap Absensi
Fitur Rekap:
- Lihat rekap kehadiran per siswa
- Pilih periode: Minggu, Bulan, Semester, atau Custom
- Export ke Excel atau PDF
- Status kehadiran otomatis (Sangat Baik, Baik, Cukup, Kurang, Buruk)
6. Logo Sekolah
Fitur Logo Sekolah:
- Admin dapat mengupload logo sekolah
- Logo akan tampil di header navbar untuk semua pengguna
- Format gambar: JPG, PNG, GIF (Max 2MB)
7. ESP32 Configuration
Setup ESP32:
- Hubungkan ke WiFi yang sama
- Pastikan RTC DS3231 terpasang
- Sensor fingerprint terhubung via multiplexer 16 channel
- Data akan tersinkronisasi ke Firebase secara real-time
