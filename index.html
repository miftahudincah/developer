📘 Panduan Lengkap Sistem Absensi

Sistem ini menghubungkan Web Dashboard dengan Alat Fingerprint (ESP32) secara langsung menggunakan Firebase Realtime Database.

1. Alur Pendaftaran & Scan Siswa Baru
  1. Siswa menyiapkan Kode Pendaftaran yang diberikan Guru.
  2. Siswa melakukan Pendaftaran melalui menu "Daftar Disini". Akun web akan terbentuk.
  3. Wajib: Setelah mendaftar, siswa belum bisa scan jarinya.
  4. Guru/Admin harus masuk ke menu Data Siswa (FP) dan memasukkan ID Fingerprint pada sensor.
  5. Setelah ID diisi, siswa baru bisa melakukan scan jari (Absen Masuk & Pulang).
2. Cara Melakukan Absensi (Scan Fingerprint)
  • ABSEN MASUK (IN): Cukup letakkan jari pada sensor saat jam masuk. Status akan berubah menjadi Hadir dan jam dicatat.
  • ABSEN PULANG (OUT): Scan jari lagi saat pulang. Sistem akan mengecek durasi waktu (default 60 menit). Jika kurang dari itu, scan akan ditolak (Peringatan).
  • Data otomatis masuk ke tabel Data Absensi di Web Dashboard.
  • Privasi Siswa: Siswa hanya bisa melihat data absensi sesuai Kelas dan Jurusan mereka sendiri.
3. Fitur Admin & Guru
  • Generate Kode: Di menu Manajemen Pengguna, klik "Generate Kode" untuk membuat kode unik satu kali pakai bagi siswa.
  • Manajemen Siswa (FP): Di menu ini, Admin/Guru bisa menambah, mengedit, atau menghapus ID Fingerprint yang terdaftar.
  • Hapus User: Admin dapat menghapus akun pengguna (Siswa/Guru) melalui tombol hapus di menu Manajemen Pengguna.
4. Profil & Pengaturan Akun
  • Klik tombol 👤 Profil Saya di pojok kanan atas.
  • Ubah Kelas/Jurusan: Di dalam profil, Anda bisa mengubah data Kelas atau Jurusan. Ini penting agar filter absensi siswa akurat.
  • Ganti Password: Gunakan menu "Ganti Password" di dalam profil untuk mengubah kunci login.
  • Ganti Foto: Klik foto profil untuk mengunggah foto baru yang akan tersimpan di ImgBB.
  • Lupa Password: Di halaman login, klik "Lupa Password?" untuk mengirim link reset email.
💡 Tips Penting:
1. Pastikan perangkat ESP32 selalu menyala ke internet agar data sinkron ke Web.
2. Data Siswa (Nama, Kelas, Jurusan) di Web harus sama persis dengan yang terdaftar di Alat Fingerprint (SD Card) agar sinkronisasi absensi jalan lancar.
