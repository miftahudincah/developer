📘 Panduan Sistem Absensi

Sistem menghubungkan Web dengan ESP32 via Firebase

1. Pendaftaran Siswa
- Guru input data siswa di menu Data Siswa (FP)
- Generate kode registrasi di menu Manajemen Pengguna
- Siswa daftar dengan kode dan ID Fingerprint
2. Absensi Fingerprint
- Scan fingerprint di sensor ESP32 untuk absen MASUK
- Scan fingerprint lagi untuk absen PULANG (setelah delay)
3. Pengaturan Kelas & Jurusan
- Admin dapat menambah/edit kelas sesuai kebutuhan
- Untuk SMK, dapat juga menambah jurusan
4. Pengumuman
- Admin/Guru dapat membuat pengumuman dengan batas waktu
- Pengumuman muncul real-time
5. Logo Sekolah
- Admin dapat mengupload logo sekolah
- Logo akan tampil di header navbar
6. Rekap Absensi
- Lihat rekap kehadiran per siswa
- Mendukung periode: Minggu, Bulan, Semester, Custom
- Bisa export ke Excel atau PDF
7. Fitur Pertemanan
- Cari teman berdasarkan email
- Kirim permintaan pertemanan
- Terima/tolak permintaan pertemanan
8. Chat & Status
- 💬 Chat: Hanya dengan teman, real-time, bisa kirim teks dan gambar
- 📸 Status: Update status yang akan hilang setelah 24 jam
9. ESP32 Configuration
- Hubungkan ke WiFi yang sama
- Pastikan RTC DS3231 terpasang
- Sensor fingerprint via multiplexer 16 channel
