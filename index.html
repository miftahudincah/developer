📘 Panduan Lengkap Sistem Absensi

Sistem ini menghubungkan Web Dashboard dengan Alat Fingerprint menggunakan Firebase Realtime Database.

1. Persiapan Awal & Login
  1. Pastikan alat Fingerprint sudah menyala dan terhubung ke internet.
  2. Buka Web Dashboard melalui browser.
  3. Gunakan akun Admin atau Guru untuk login.
💡 Default Akun Admin:
Email: admin@sekolah.com
Password: admin123
2. Registrasi Fingerprint Siswa Baru
  1. Tekan tombol REGISTER/ENROLL pada alat fingerprint.
  2. Tempelkan jari siswa pada sensor fingerprint (2 kali).
  3. Tunggu notifikasi "Sukses" dari alat.
  4. Edit data siswa di menu Data Siswa (FP).
3. Generate Kode Pendaftaran

Untuk Siswa:

  1. Klik tab Manajemen Pengguna.
  2. Pilih "Untuk Siswa (Pilih Nama)".
  3. Pilih siswa dari dropdown, klik Generate Kode.
  4. Berikan kode ke siswa untuk daftar akun.

Untuk Guru:

  1. Pilih "Untuk Guru", klik Generate Kode.
  2. Berikan kode ke calon guru.
⏱️ Kode berlaku 5 jam dan satu kali pakai.
4. Proses Absensi (Scan Fingerprint)
  • MASUK: Scan jari → status Hadir
  • PULANG: Scan jari lagi (minimal 60 menit) → status Pulang
5. Manajemen Pengguna
  • Ubah Role: Klik dropdown Role di tabel Daftar Pengguna
  • Hapus User: Klik tombol 🗑️ di kolom Aksi
6. Pengaturan Sistem (Admin Only)
  • Ubah Nama Sekolah: Isi field Nama Sekolah → Simpan
  • Reset Data: Klik Reset Firebase (Hati-hati!)
7. Fitur Teman Sekelas (Siswa)
  • Siswa dapat melihat teman sekelas (kelas & jurusan sama)
  • Status absensi hari ini (Hadir/Belum Absen)
8. Ekspor Data & Simulasi
  • Download Excel: Klik tombol 📥 Download Excel
  • Simulasi Scan (Admin): Klik 📷 Simulasi Scan FP
💡 Tips Penting:
  • Backup data rutin dengan ekspor ke Excel
  • Data siswa di menu Data Siswa (FP) adalah sumber utama sinkronisasi
  • Gunakan filter Kelas/Jurusan untuk memudahkan pencarian
❓ FAQ

Q: Data absensi tidak muncul?
A: Cek koneksi internet alat fingerprint dan filter tanggal.

Q: Kode pendaftaran tidak valid?
A: Pastikan kode belum kadaluarsa (5 jam).

Q: Siswa lupa password?
A: Klik Lupa Password di halaman login.
